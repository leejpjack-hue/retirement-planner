# 退休規劃應用 - Retirement Planner

## 選擇既設計
- **Style**: Student-friendly (Anti-AI Slop)
- **Font**: Baloo 2 + Nunito
- **Color**: Teal/Green primary

## 功能
1. 選擇目前城市
2. 選擇退休城市  
3. 輸入年齡
4. 生活方式問卷 (食/旅行/交通/興趣)
5. 計算退休所需金額
6. 顯示建議

## 技術
- React 18 + TypeScript
- Vite
- CSS Modules

## 設計原則 (anthropic-frontend-design skill)
- 避免 generic AI generated look
- 用独特既 typography
- 简洁但有特色既 color palette
