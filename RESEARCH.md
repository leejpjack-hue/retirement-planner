## 5. 計算公式

### 4% Rule
```
退休總額 = (每月支出 × 12) / 0.04
```

### 通脹調整
```
未來每月支出 = 現時支出 × (1 + 通脹率)^(退休年齡 - 現在年齡)
```

### 每月儲蓄
```
每月儲蓄 = 總額 × (回報率/12) / ((1 + 回報率/12)^(年數×12) - 1)
```
